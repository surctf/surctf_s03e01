<h1 class="font-pt-mono text-xl font-bold text-black">Управление</h1>
Стрелки - передвижение<br>
Z - прыжок<br>
X - ускорение (заряжаемое)<br>
R - рестарт уровня<br>
Alt+Enter - полный экран<br><br />
